Simulação 3D de Física para Jogos

Desenvolvimento de uma simulação gravitacional interativa em 3D, implementada em JavaScript com o uso de p5.js e WebGL. A simulação recria cenários complexos de interação entre partículas sujeitas a forças gravitacionais, proporcionando visualizações dinâmicas e realistas.

Para otimizar o desempenho em sistemas com milhares de partículas, o projeto emprega o algoritmo de Octree, em conjunto com a aproximação de Barnes-Hut, reduzindo significativamente o número de cálculos necessários para determinar as forças gravitacionais. Essa abordagem permite um equilíbrio entre precisão e eficiência, essencial para simulações em tempo real.

Além disso, a aplicação oferece recursos visuais interativos, como zoom, rotação e ajustes de parâmetros físicos, permitindo explorar os efeitos da gravidade em diferentes condições. Este projeto destaca-se como uma poderosa ferramenta educacional e de experimentação para simulações de física aplicada.

Flappy Bird AI

Criação de uma inteligência artificial para jogar o famoso jogo Flappy Bird, utilizando redes neurais e algoritmos genéticos. O projeto emprega uma abordagem inovadora onde o algoritmo genético é responsável por evoluir os pesos da rede neural ao longo de várias gerações. Cada indivíduo (rede neural) é avaliado com base no desempenho dentro do ambiente do jogo, recebendo pontuações proporcionais à distância percorrida e à quantidade de obstáculos superados.

As melhores redes são selecionadas para reprodução, combinando e mutando os pesos para gerar uma nova população. Esse processo iterativo permite que a IA melhore progressivamente suas habilidades sem supervisão direta. A simulação do ambiente foi implementada em p5.js, enquanto os cálculos e operações do algoritmo genético foram otimizados com bibliotecas de JavaScript para garantir desempenho.

Ligue 4

Desenvolvimento de uma versão interativa do clássico jogo Ligue 4, totalmente implementada utilizando HTML, CSS e JavaScript. O projeto combina uma interface intuitiva e responsiva com uma inteligência artificial que desafia os jogadores a cada partida.

A IA utiliza o algoritmo Minimax, uma abordagem de busca baseada em teoria dos jogos, para calcular os melhores movimentos em tempo real. Com a ajuda de técnicas como poda alfa-beta, o desempenho do algoritmo é otimizado, permitindo decisões rápidas e eficientes, mesmo em estados avançados do jogo.

O jogo oferece uma experiência envolvente para jogadores de todos os níveis, desde iniciantes até os mais experientes. Recursos adicionais incluem a visualização do tabuleiro em diferentes resoluções e ajustes dinâmicos na dificuldade, garantindo diversão e aprendizado.
